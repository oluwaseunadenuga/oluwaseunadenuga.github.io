## System Architecture ### Frontend Architecture **Framework**: React 18 with TypeScript and Vite as the build tool - Single-page application (SPA) using Wouter for client-side routing - Component-based architecture with reusable UI components - Responsive design optimized for desktop and mobile devices **UI Framework**: Tailwind CSS with shadcn/ui component library - Modern design system with consistent styling patterns - CSS variables for theming support (light/dark modes) - Radix UI primitives for accessible, unstyled components - Custom component variants using class-variance-authority **State Management**: React Query (TanStack Query) for server state management - Optimistic updates and caching strategies - Error handling and retry mechanisms - Background refetching and synchronization ### Backend Architecture **Server Framework**: Express.js with TypeScript - RESTful API design with /api route prefix - Middleware for request logging and error handling - Development-optimized with Vite integration for HMR **Data Layer**: Drizzle ORM with PostgreSQL - Type-safe database operations with schema validation - Migration support through drizzle-kit - Neon Database serverless PostgreSQL integration **Storage Strategy**: Hybrid approach with MemStorage for development - In-memory storage implementation for rapid development - Interface-based design allowing easy database integration - CRUD operations for users and projects entities ### Development Environment **Build System**: Vite with custom configuration - Hot module replacement (HMR) for development - Optimized production builds with code splitting - Path aliases for clean import statements **Development Tools**: - TypeScript for type safety across the entire application - ESLint and Prettier for code quality (implied by structure) - PostCSS with Tailwind CSS for styling pipeline ## External Dependencies ### Database Services - **Neon Database**: Serverless PostgreSQL database hosting - **Drizzle ORM**: Type-safe database toolkit and query builder - **connect-pg-simple**: PostgreSQL session store for Express sessions ### UI and Styling - **Tailwind CSS**: Utility-first CSS framework for styling - **Radix UI**: Comprehensive collection of accessible React components - **Lucide React**: Modern icon library for React applications - **shadcn/ui**: Pre-built component system built on Radix UI ### Development and Build Tools - **Vite**: Fast build tool and development server - **TypeScript**: Static type checking for JavaScript - **React**: Frontend library for building user interfaces - **React Query**: Server state management and data fetching ### External Portfolio Links - **Tableau Public**: Interactive dashboard hosting for data visualizations - **Google Drive**: Document and report storage for project deliverables - **Unsplash**: High-quality stock photography for project imagery ### Additional Integrations - **React Hook Form**: Form state management with validation - **Zod**: Runtime type validation and schema parsing - **date-fns**: Modern date utility library for JavaScript - **Wouter**: Minimalist routing library for React application
